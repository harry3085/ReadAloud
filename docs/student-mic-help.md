# 큰소리 영어 — 녹음숙제·말하기 시험 안 될 때

녹음숙제나 말하기 시험을 시작하려는데 "마이크 권한이 필요해요" / "음성 인식이 안 돼요" / "답을 말하기도 전에 넘어가요" 같은 증상이 있으면 아래 안내를 따라 해보세요.

대부분 **폰의 마이크 권한** 또는 **인터넷 환경**, **OS·브라우저 버전** 문제입니다.

---

## 빠르게 확인할 것 (1~2분 안에 해결되는 경우)

1. **wifi 끄고 LTE 로 바꿔서 다시 해보기**
   - 학원·학교 wifi 가 일부 음성 서비스를 막아두는 경우가 있어요. LTE 로 잘 되면 wifi 문제입니다.
2. **브라우저 새로고침**
   - 화면 상단을 아래로 당겨서 새로고침.
3. **다른 브라우저로 열어보기**
   - iPhone: Safari → Chrome (또는 반대로)
   - Android: Chrome → Samsung Internet (또는 반대로)

위 3가지로 해결되면 추가 작업 불필요. 그래도 안 되면 아래 OS 별 안내를 보세요.

---

## iPhone (아이폰) 사용 학생

### 1. iOS 버전 확인 — **14.5 이상 필수**

말하기 시험은 iOS 14.5 이상에서만 작동해요. (녹음숙제는 이보다 낮은 버전도 가능)

- **설정** → **일반** → **정보** → **iOS 버전** 확인
- 14.5 미만이면: **설정** → **일반** → **소프트웨어 업데이트** → 업데이트 실행
- 폰이 너무 오래되어 업데이트가 안 되면 다른 폰으로 시도

### 2. Safari 마이크 권한 허용

1. **설정** 앱 열기
2. **Safari** 탭으로 이동
3. 아래로 스크롤 → **마이크** 메뉴
4. **"raloud.vercel.app"** 항목이 **"허용"** 으로 되어 있는지 확인
5. **"거부"** 로 되어 있으면 → **"허용"** 으로 변경

또는 사이트별 설정:

1. Safari 에서 **raloud.vercel.app** 접속
2. 주소창 왼쪽 **"aA"** 또는 자물쇠 아이콘 탭
3. **웹사이트 설정** → **마이크** → **허용**

### 3. 홈 화면 PWA 로 사용 중인 경우

- **설정** → **개인정보 보호 및 보안** → **마이크** → "큰소리 영어" 항목 허용

[학원장님: iPhone 권한 화면 캡처 자리]

---

## Android (안드로이드) 사용 학생

### 1. Chrome 마이크 권한 허용

1. **설정** 앱 열기
2. **앱** → **Chrome** → **권한** → **마이크** → **허용**

또는 사이트별 설정:

1. Chrome 에서 **raloud.vercel.app** 접속
2. 주소창 왼쪽 자물쇠 아이콘 탭
3. **권한** → **마이크** → **허용**

### 2. Samsung 갤럭시 — Bixby 가 아닌 Google 음성 입력으로 변경

말하기 시험은 **Google 음성 인식 서비스**를 사용해요. 갤럭시 기본 음성 입력이 Bixby 로 되어있으면 안 됩니다.

1. **설정** → **일반** → **언어 및 입력**
2. **음성 입력 서비스** 또는 **온스크린 키보드** → **기본 음성 입력**
3. **Google 음성 입력**으로 변경

[학원장님: Samsung 설정 캡처 자리]

### 3. Google 앱 설치·활성화 확인

- 플레이스토어에서 **Google 앱** 검색 → 설치되어 있는지 / 비활성화되어 있지 않은지 확인
- 비활성화된 상태면 → 활성화

---

## 그래도 안 될 때

위 단계 다 해봤는데 여전히 안 되면 학원장님에게 다음 정보를 알려주세요:

- **폰 종류**: iPhone / 갤럭시 / 기타 (모델명까지)
- **OS 버전**: iOS 17.x / Android 14 등
- **브라우저**: Safari / Chrome / Samsung Internet 등 + 버전
- **wifi 인가 LTE 인가**
- **어느 시험에서 발생** (녹음숙제 / 말하기 시험 / 둘 다)
- **정확한 화면 안내문 캡처**

---

## 자주 묻는 질문

**Q. 녹음숙제는 되는데 말하기 시험만 안 돼요.**

→ 녹음숙제와 말하기 시험은 다른 기술을 사용해요. 녹음숙제는 폰 안에서 직접 녹음하지만, 말하기 시험은 마이크 음성을 Google 음성 서버로 보내 글자로 변환해요. 그래서 인터넷이 차단되거나 Google 음성 서비스가 꺼져있으면 말하기 시험만 안 됩니다.

**대처**: 위의 "wifi → LTE 전환" 및 "Samsung 음성 입력 → Google" 안내 확인.

**Q. 시험 화면에서 답을 말하기도 전에 자동으로 다음 문제로 넘어가요.**

→ 마이크 권한이 거부되어 있거나, 음성 인식 서비스가 즉시 실패해서 자동으로 오답 처리된 거예요. 폰 설정에서 권한 허용 후 다시 시도하세요.

**Q. "이 브라우저는 음성 인식을 지원하지 않습니다" 라고 떠요.**

→ 브라우저 버전이 너무 낮은 경우입니다. iOS 라면 14.5 이상 업데이트, Android 라면 Chrome 을 최신 버전으로 업데이트하세요.

**Q. 권한을 허용했는데도 안 돼요.**

→ 브라우저를 완전히 종료한 후 다시 켜서 사이트 접속해보세요. PWA (홈 화면 아이콘) 로 사용 중이면 PWA 도 한 번 닫고 다시 열어주세요.
