# 큰소리 영어 — 알림 끄기·켜기 안내

학원에서 보내는 푸시 알림(숙제 안내·시험·일정 등) 을 받지 않으려면 폰 설정에서 알림을 끄세요.
앱이나 학원에서 차단할 수 있는 게 아니라 **본인 폰의 OS 설정**에서만 가능해요.

알림을 다시 받고 싶을 때도 같은 메뉴에서 켜면 됩니다 (앱 재설치·재로그인 불필요).

---

## 📱 iPhone

### Safari 또는 Chrome 으로 사이트 사용 중일 때

1. 폰 **설정** 앱 열기
2. **Safari** (또는 **Chrome**) 탭으로 이동
3. **고급** → **웹사이트 데이터** → "raloud.vercel.app" 찾기
4. 또는 **알림** 메뉴에서 사이트 단위 알림 차단

> iPhone Safari 는 OS 14.5 이상부터 웹 푸시 지원. 그 이전 버전은 처음부터 알림 안 옴.

### 홈 화면에 추가한 PWA 인 경우

1. 폰 **설정** 앱 열기
2. **알림** 메뉴
3. 앱 목록에서 **"큰소리 영어"** 찾기
4. **알림 허용** 토글 OFF

---

## 🤖 Android

### Chrome 일반 탭으로 사용 중일 때

1. Chrome 열고 **raloud.vercel.app** 접속
2. 주소창 좌측 **🔒 자물쇠** 또는 **ⓘ** 아이콘 탭
3. **사이트 설정** → **알림** → **차단**

또는:

1. Chrome **⋮ 메뉴** → **설정** → **사이트 설정** → **알림**
2. 목록에서 **raloud.vercel.app** 찾아 **차단**

### 홈 화면에 설치한 PWA 인 경우

1. 폰 **설정** 앱 열기
2. **앱** 또는 **앱 관리**
3. 목록에서 **"큰소리 영어"** 찾기
4. **알림** → **모든 알림 표시** 토글 OFF

---

## 다시 알림 받고 싶을 때

같은 메뉴에서 알림을 **다시 허용** 으로 켜면 됩니다.
앱 재설치·로그인 다시 안 해도 자동으로 알림이 다시 도착해요.

---

## 자주 묻는 질문

**Q. 학원장이 알림을 끄게 해줄 수는 없나요?**
A. 학원 측에서는 학생별 알림 끄기 기능을 별도로 제공하지 않아요. 폰 OS 설정에서만 끌 수 있습니다.

**Q. 알림 끄면 앱 안의 알림함도 비워지나요?**
A. 아니요. OS 푸시만 안 떠요. 앱에 들어가면 학원장이 보낸 메시지는 알림함(🔔)에서 그대로 볼 수 있어요.

**Q. 다른 폰(예: 부모님 폰) 도 알림이 와요. 따로 끄려면?**
A. 같은 학생 ID 로 여러 폰에 로그인하면 모든 폰이 알림을 받습니다. 받지 않으려는 폰에서 위 안내대로 OS 알림을 끄거나, 그 폰만 학생앱에서 로그아웃하세요.

**Q. 일정 시간만 끄고 싶어요 (예: 학원 시간 외).**
A. 폰의 **방해 금지 모드** (집중 모드 / 야간 모드) 를 활용하시면 됩니다. 시간 지나면 자동 해제 됩니다.
